# NYC Airbnb Open Data — EDA

Exploratory data analysis of ~49,000 NYC Airbnb listings from 2019.

## Key Findings
- Manhattan median price (150 USD) is nearly double Brooklyn (90 USD)
- Entire homes cost ~2x private rooms (160 vs 70 USD median)
- Price has almost no correlation with availability or review count
- Airbnb grew explosively in NYC from 2015 onwards

## Tools Used
Python, pandas, numpy, matplotlib, seaborn

## Dataset
[NYC Airbnb Open Data — Kaggle](https://www.kaggle.com/datasets/dgomonov/new-york-city-airbnb-open-data)
